# Fast API Game Server - RPG Edition

## Project Overview
The Fast API Game Server is a multiplayer RPG (Role-Playing Game) backend application built with FastAPI, Firebase, and Pydantic. Players can create characters, engage in PvP matchmaking battles, and progressively upgrade their character stats based on victories.

## Tech Stack

### Backend
- **FastAPI** - Modern, fast web framework for building APIs
- **Firebase** - Real-time database and authentication
- **Pydantic** - Data validation using Python type annotations

### Frontend
- **React** - JavaScript library for building user interfaces
- **HTML/CSS/JavaScript** - Core web technologies
- **Bootstrap** - Responsive CSS framework for UI components

## Features

### User Management
- **Login/Register** - Secure user authentication and account creation
- **User Profiles** - Manage user accounts and game progress

### Character Management
- **Create Characters** - Players can create multiple game characters
- **Character Stats** - Each character has basic stats (Defense, Attack)
- **Delete Characters** - Remove characters from the game

### Game Mechanics
- **Matchmaking System** - Find opponents for PvP battles
- **Combat System** - Battle mechanics where stats influence win probability
- **Stat Progression** - Winners can upgrade one of two stats (Defense or Attack)
- **Probability-Based Outcomes** - Players with better stats have higher chances of winning
